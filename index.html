About Me

Hi! My name is Aileen Denton and I'm currently a Creative Computing Student at IADT. This is my portfolio of work completed in the Creative Coding module. Id used p5.js to create generative artworks while learning about colour theory and composition. In each of the cards below you can see my work, view and interact with some of them. They will also display a description of methods and code I used to create them.
