Random Point Bounce

Feb 2020

Description

This task involves using random angles that are controlled through if statements to detect specific canvas edges and bounce off depending on the edge.

This skecth uses an if statement to detech the x position and y position of the point on the canvas. If the x or y position returned is equal to the 0, or the width or height, the angle variable that the point moves changes.

if (xPos < 0 || xPos > width || yPos < 0 || yPos > height) {
angle = getRandomAngle(xPos, yPos)
}

The angles are determined using a function called getRandomAngle(). This function returns the x and y position of the point. It also holds a series of if statements to determine the position of the point and apply a specific angle for the point to travel upon.

function getRandomAngle(_x, _y) {
let randomAngle = (floor(random(-angleCount, angleCount)) + 0.5) * 90 / angleCount;
//console.log(randomAngle);
if (_y < 0) {
return randomAngle
}
if (_y > height) {
return randomAngle + 180;
}
if (_x > width) {
return randomAngle + 90;
}
if (_x < 0) {
return randomAngle + 270;
}
}
