Understanding Atan2

Oct 2020

Description

Like the Rotating Rectangle task, we used atan2 to create an interactive piece in which an object's rotations will be determined by the position of the cursor.

The color of these shapes are defined by the number of shapes established in the script. The color is incremented as each shape is drawn, using the RGB colorMode and defining the max values for Red, Green and Blue as the number of shapes.

colorMode(RGB,numOfShapes,numOfShapes,numOfShapes);

This exercise used ellipses, with smaller ones inside them to create a mesh-like movement. The smaller circles will rotate around their translation origin point according to the position of the mouse.

//Calculate the angle between the mouse and object
let rotValue = atan2(mouseY-transY, mouseX-transX);
rotate(rotValue);
noStroke();
fill(col);
ellipse(0, 0, shapeSize, shapeSize);
fill(0);
ellipse(10,0,shapeSize/4,shapeSize/4);
pop();
