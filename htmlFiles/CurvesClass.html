Curves Class

Feb 2020

Description

This task was used to create a circle using the curveVertext function. It also required interactivity and the use of objects and classes.

This composition uses similar functionality to the 'curves function' piece, in that it uses cosine and sine to calculate angles for ther circle created. The results are then passed into a vector which is pushed into an array of vectors. These vectors will hold the x and y co-ordinates of the circle.

for(let i=0; i<=numOfSeg; i++){
let vx = cos(angle*i)*r;
let vy = sin(angle*i)*r;
vPoints.push(createVector(vx,vy));
}

The Curve objects created takes three inputs, the radius, the point radius and the vPoints. The vPoints are the created vectors that define each point that outlines the circle.

class Curve{
//Used for passing through radius for overall circle, point radius and the array points
constructor(r_, pointRadius_, vPoints_){
this.r = r_;
this.pointRadius = pointRadius_;
this.vPoints = vPoints_;
}

Once the curve is rendered, it creates a line from each point in the array to the point after it.

for(let i = 0; i<this.vPoints.length - 1; i++){
stroke(255 , 50);
strokeWeight(2);
//Adding one to the vPoints because of the array index starting at 0
line(this.vPoints[i].x, this.vPoints[i].y, this.vPoints[i + 1].x, this.vPoints[i + 1].y);
}
}
