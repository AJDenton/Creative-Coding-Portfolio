Growth

Feb 2020

Description

This task involved using agency to develop a growing simulation. In doing so, an array of points is created. Once each point is drawn, it will move towards the closest point in the array. Once the points make contact, another point will be drawn.

The inital point is established by defining the destination as the middle of the canvas. Two functions are created in order to check the distance between two points, and the move the shape. The shape that will be drawn is drawn using a random position on the x and y axis.

points.push(createVector(width/2, height/2));
//Setting the destination to the first array value
destination = points[0];
//Creating a shape that will attach to the root shape. This is assigned to a random position on the canvas
shapeToMove = createVector(random(0,width), random(0,height));

The check distance function will check through the points array in order to find the shortest distance between the point drawn and the points checked. It will then return the closest distance. This value is then assigned to the 'destination' variable, which is used in the 'moveShape()' function.

points.forEach(element => {
if(shapeToMove.dist(element) < maxDist){
//assign the maxDist to the lowest value in the array
maxDist = shapeToMove.dist(element);
//closest distance will be the lowest distance value in the array when compared to the shapeMove
closestDist = element;
}
});
//Return the closestDist value
return closestDist;

The moveShape() function uses the destination assigned to move the points towards their required position. This uses a p5.Vector.lerp function in order to move the drawn point to the destination.

shapeToMove = p5.Vector.lerp(shapeToMove, destination, amount);
