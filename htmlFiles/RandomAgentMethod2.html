Random Agent Method 2

Feb 2020

Description

This task uses a series of arrays to determine the tendency of random movement of a shape. This uses lines to create a randomised composition that stems from the center of the canvas, with a tendency to move then opposing end of the line downwards.
let optionsX = [-1, -1, 0, 1, 1];
let optionsY = [-1, 1, 1, 1, 1];

Similarly to the first Random Agent composition, this code uses similar movement theory. This code however, uses a series of arrays, from which the code obtains a random array value and uses this with the stepsize to establish a movement.
let moveX = optionsX[floor(random(0, optionsX.length))] * stepSize;
let moveY = optionsY[floor(random(0, optionsY.length))] * stepSize;

This code also uses if statements to check whether the moving shape has reached the edge of the canvas. In doing so, the shape will disappear and respawn on the opposing side of the canvas. For example;
if (xPos > width) { xPos = 0; } else if (xPos < 0) { xPos = width; }
This code uses two line shapes to illustrate and offset shadow-like image. Because of this, each line has it's own movement and arrays to select from. The only difference is that the second drawn line uses an offset.
