Random Agent

Feb 2020

Description

This task uses a combination of mapping and randomness in order to create a randomly generating pattern.
This code uses random ranges to move a single ellipse shape on both the X and Y axis. Firstly the direction is established, then the movement is established using the direction and stepSize.
let directionX = round(random(-1,1));
let directionY = round(random(-1,1));
let moveX = directionX * stepSize;
let moveY = directionY * stepSize;
By doing this, the ellipse is able to move horizontally, vertically and diagonally.
This is wrapped inside a for loop that uses the "speed" variable. The speed is mapped to the mouseX position on the canvas.
let speed = map(mouseX,0,width,2,40);
This mapped variable will determine how fast the ellipse shape travels around the canvas.
